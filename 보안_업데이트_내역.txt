========================================
  보안 기능 업데이트 완료
========================================

[적용된 보안 기능]

1. ✅ 로그인 시도 제한 (브루트포스 방어)
   - 5분 내 5회 로그인 실패 시 해당 IP를 15분간 차단
   - 남은 시도 횟수를 사용자에게 표시
   - 차단된 IP는 자동으로 시간 경과 후 해제

2. ✅ 접속 로그 기록
   - 모든 로그인 시도 기록 (성공/실패)
   - 파일 다운로드 기록
   - 폴더 다운로드 기록
   - IP 주소, 사용자명, 시간, 행동 기록
   - 서버 콘솔에 실시간 로그 출력
   - 최근 1000개 로그 메모리 유지

[로그 예시]
[로그] 2025-11-09 12:30:15 | 192.168.1.100 | seungjjj | 로그인 성공 | 
[로그] 2025-11-09 12:30:45 | 192.168.1.100 | seungjjj | 파일 다운로드 | document.pdf
[로그] 2025-11-09 12:31:20 | 192.168.1.100 | seungjjj | 폴더 다운로드 | 사진모음

[기존 보안 기능]
✅ Cloudflare Tunnel (HTTPS 암호화, DDoS 방어)
✅ 비밀번호 해시화 (werkzeug)
✅ 공유 폴더 외부 접근 차단
✅ 읽기 전용 (다운로드만 가능)

========================================
  EXE 파일 빌드 방법
========================================

1. build_pyqt_all.bat 실행
2. 자동으로 클라이언트와 서버 exe 생성
3. 완성된 파일 위치: dist 폴더
   - Woori_Client_PyQt.exe (클라이언트)
   - Woori_Server_PyQt.exe (서버)

========================================
  테스트 방법
========================================

[BAT 파일로 테스트]
1. test_server_pyqt.bat - 서버 실행
2. test_client_pyqt.bat - 클라이언트 실행

[보안 테스트]
1. 서버 실행
2. 클라이언트에서 잘못된 비밀번호로 5번 시도
3. "로그인 실패 횟수 초과" 메시지 확인
4. 서버 콘솔에서 로그 확인

========================================
  주의사항
========================================

⚠️ 여전히 남아있는 약점:
- 클라이언트에 비밀번호 평문 저장 (편의성을 위해)
- 세션 타임아웃 없음 (한번 로그인하면 계속 유효)

🔒 추가 보안이 필요한 경우:
- IP 화이트리스트 설정
- 세션 타임아웃 추가
- 클라이언트 비밀번호 암호화
